## Minikube Lab

#### Start Minikube

Creates 1 control-plane and 1 worker node (minikube and minikube-m02)

```
minikube start --nodes 2
```

#### Stop Minikube

Preserves cluster state and data (can be resumed later)

```
minikube stop
```

#### Delete Minikube

Destroys all Minikube VMs/containers and cluster data

```
minikube delete
```
